What is Climbing?

Climbing is a sport which originated with the purpose to mountaineer. Later, in the 1970s and 80's, it became more common, and climbing shifted to be about climbing cliff faces in places like Yosemite, Moab, Red River Gorge, etc. Now, climbing has broadened to be indoors, in gyms as well, in a competitive form, but many participate in all disciplines.
